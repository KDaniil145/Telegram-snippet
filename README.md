# Telegram Snippet - VS Code Plugin

**Автор:** Корчагин Даниил Сергеевич  
**Группа:** М3102

## Описание
Плагин для быстрого сохранения выделенного кода в файл и автоматического открытия Telegram для отправки.

## Использование
1. Выделите код в VS Code
2. Нажмите `Ctrl+Shift+T`
3. Файл сохраняется в `Downloads/snippet.cpp`
4. Автоматически открывается Telegram Desktop
5. Перетащите файл в диалог
