# Описание функций

## activate(context)
**Назначение:** Активация плагина в VS Code

**Параметры:**
- context - контекст расширения VS Code

**Пример вызова:**
```typescript
// Активируется автоматически при первом использовании команды плагина
activate(extensionContext); 
```

## Основаная логика плагина
Расположена в  `registerCommand`  

**Назначение:** Основная функция сохранения snippet'a

**Логика работы:**
1. Проверяет наличие активного редактора
2. Получает выделенный текст  
3. Сохраняет в файл Downloads/snippet.cpp
4. Запускает Telegram Desktop

**Пример вызова:**
вызывается автоматически при нажатии `Ctrl+Shift+T`

## deactivate()
**Назначение:** Деактивация плагина при выгрузке  
**Пример вызова:**
```typescript
// Вызывается автоматически VS Code при выгрузке плагина
deactivate();
```

## Используемые модули

### Встроенные модули Node.js
- `child_process.exec` - для запуска внешних приложений (Telegram)
- `path.join` - для работы с путями файловой системы  
- `os.homedir` - для получения пути к домашней директории пользователя

### VS Code API
- `vscode.commands` - система команд расширения
- `vscode.workspace.fs` - файловая система workspace
- `vscode.window` - работа с интерфейсом и уведомлениями
